# 中医中药病证单位教材编写平台

## 项目概况
- **8 大院系**：中医/中药/针灸/推拿/骨伤/五官口腔/中医智能/中医管理
- **120 门学科** → ~134 种教材
- **四层架构**：L1 行业规划主编 → L2 领域主编 ×8 → L3 学科主编 ×40 → L4 章节作者 ×800
- **最小编写原子**：病证单位（病侧西医分子靶点 + 证侧中医证候场论）
- **六阶段流程**：需求分析 → 体系设计 → 内容编写 → 审核评阅 → 教学应用 → 修订迭代
- **CI/CD 人机互动**：Hermes Agent 原始编写 → 人类三方（审稿/编写/使用）审核反馈
